Create a user account for accessing the Airflow web interface:

airflow users create \
    --username admin \
    --password admin_password \
    --firstname Admin \
    --lastname User \
    --role Admin \
    --email admin@example.com



Start the Airflow web server and scheduler:

airflow webserver --port 8080 &  # Starts the webserver on http://localhost:8080
airflow scheduler &               # Starts the scheduler in the background
